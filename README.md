# Ruby Basics Training

## Overview

This project contains a series of exercises to **learn and practice the basics of Ruby**.  
The goal is to get familiar with the following concepts:

- User input  
- Variables and basic data types  
- Loops (`times`, `while`, `upto`, `downto`)  
- Conditions (`if`, `unless`, `even?`)  
- String manipulation (`String#*`, interpolation, `format`)  
- Arrays (`Array`, adding elements, filtering)  
- Creating simple visual patterns (pyramids)

Each exercise is a separate Ruby file named `exo_X.rb`.

---

## Concepts Learned

| Concept             | Commands / Methods Used                                        |
| ------------------- | -------------------------------------------------------------- |
| User input          | `gets`, `chomp`, `to_i`                                       |
| Loops               | `times`, `while`, `upto`, `downto`, `each`                    |
| Conditions          | `if`, `unless`, `even?`                                       |
| Arrays              | `Array`, `<<`, `each`                                         |
| Strings             | `String#*`, `#{}`, `format`                                   |
| Output              | `puts`, `print`                                               |
| Basic math          | `+`, `-`, `*`                                                 |

---

## Objectives

- Master loops and conditions in Ruby  
- Manipulate strings and arrays  
- Understand string formatting, interpolation, and repetition  
- Produce simple visual results (pyramids)  
- Develop step-by-step programming logic
